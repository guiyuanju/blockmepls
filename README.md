# Block Me PLS

## Installation

```bash
go install https://github.com/guiyuanju/blockmepls
```

## Usage

```bash
# block
blockmepls -sites="x.com,instagram.com"

# reset
blockmepls -reset
```

```
Usage of blockmepls:
  -reset
        reset the block list
  -sites value
        the sites to be blocked, comma separated
exit status 1
```
